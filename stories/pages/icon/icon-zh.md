## API

| 参数   | 类型    | 默认值 | 说明                                      |
| :----- | :------ | :----- | :---------------------------------------- |
| spin   | boolean | false  | 是否有旋转动画                            |
| rotate | number  | -      | 图标旋转角度                              |
| name   | string  | -      | 图标名，可以在 Icon List 中查看有哪些图标 |
